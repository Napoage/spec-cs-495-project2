Main Menu Instructions

On this page, you will have three main options to navigate elsewhere and a LOGOUT button for secure exit. Below are the available options and their corresponding functions:

Utilities

This section provides essential tools for managing the device and system operations:

  • SITE INFO: View or update site-specific information.
  • LIVE VIDEO: Access a live feed from the device camera.
  • SAVE DATA: Save collected data for further analysis.
  • VIEW LOGS: Review system logs for diagnostics or troubleshooting.
  • CHECK DISK SPACE: Check remaining disk space and time till it runs out. .
  • REBOOT SYSTEM: Restart the system if needed.

Setup and Calibrations

Use this section to configure and prepare the system for PIV operations:

  • TRAPEZOID CALIBRATION: Define the area of interest for PIV analysis.
  • PIV PARAMETERS: Set key parameters for PIV calculations.
  • MASKING OPTION: Exclude non-relevant areas from the PIV analysis to improve accuracy.
  • CAMERA PARAMETERS: Adjust camera based parameters, should only be set once.

Use the LOGOUT button to securely exit the application when your session is complete.
